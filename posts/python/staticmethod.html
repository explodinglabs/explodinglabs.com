Don't use @staticmethod

Beau Barker, Jan 25, 2021. Updated Sep 12, 2025. ✎

In Python, don’t use the @staticmethod decorator.

A static method is just a function that doesn’t use the class or object it’s contained in.

In other words, it doesn’t need to be inside a class, so write a plain function instead.
