Python's collections

Beau Barker, Oct 13, 2018. Updated Jul 25, 2025. ✎

These are the most commonly used collections:

  Immutable Mutable
Sequence tuple, str, bytes list, bytearray
Set frozenset set
Mapping   dict

Or shown in a heirarchy:

Sequence

  • tuple
  • str
  • ByteString
    • bytes
    • bytearray
  • MutableSequence
    • list
    • bytearray

Set

  • frozenset
  • MutableSet
    • set

Mapping

  • MutableMapping
    • dict
