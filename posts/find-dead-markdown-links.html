Find Dead Markdown Links

Beau Barker, Oct 29, 2025. Updated Oct 28, 2025. ✎ 
npm install -g markdown-link-check

.mlc.json

{
  "ignorePatterns": [
    { "pattern": "^http" },
    { "pattern": "^mailto:" }
  ],
  "replacementPatterns": [
    { "pattern": "^\\./", "replacement": "" }
  ]
}

find . -name '*.md' -print0 | xargs -0 -n1 markdown-link-check -q --config .mlc.json
