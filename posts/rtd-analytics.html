How to add Google Analytics to Readthedocs?

Beau Barker, Aug 2, 2016. Updated Jul 15, 2025. ✎

To add your GA tracking ID to readthedocs:

  1. Login to readthedocs.org, and browse to your project
  2. Go to Admin > Advanced Settings
  3. Enter the tracking ID into “Analytics code”.

If you use a custom theme, you may need to add the code into conf.py. For example, see analytics_id in Alabaster’s theme options.
