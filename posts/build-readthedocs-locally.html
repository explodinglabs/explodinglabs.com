How to test Readthedocs documentation locally?

Beau Barker, Aug 6, 2021. Updated Nov 5, 2025. ✎

Install Sphinx.

pip install sphinx

Build your docs:

$ sphinx-build -a doc /tmp/mydocs

Open the html file in your browser, for example:

file:///tmp/jsonrpcserver/index.html
