SuperStack is Live

Beau Barker, Jul 25, 2025. Updated Oct 28, 2025. ✎

I built SuperStack for devs who want a minimal backend to start with, and full control to build it up as they go.

It comes with just the essentials:

  • Caddy — API gateway
  • PostgREST — REST API (and RPC)
  • Postgres — database

It’s a minimal stack. It’s not “all batteries included.” No mystery containers you’ll never touch. Zero bloat.

The idea is:

  1. Start lean – no heavy services you don’t need.
  2. Build as you grow – add authentication, messaging, Stripe, by following the Wiki.

I chose Caddy after years of running nginx and it was the right move.

If you’re a SaaS founder, indie hacker or backend engineer who values control, I’d love your feedback.
