    SuperStack uses Docker, so make sure Docker is installed before you begin.

    1. Get SuperStack

    Option 1: Use the Template (Recommended)

    The easiest way to get started:

    1. Click Use this template on GitHub.
    2. Create a new repository (e.g. myapp) from the template.
    3. Clone it to your machine:
    git clone https://github.com/yourname/myapp.git
cd myapp

    Option 2: Clone and Track Upstream (Advanced)

    If you want to keep SuperStackâ€™s Git history and pull upstream changes later:

    git clone https://github.com/explodinglabs/superstack.git myapp
cd myapp
git remote rename origin upstream
git remote add origin https://github.com/yourname/myapp

    You can now pull upstream changes with:

    git pull upstream main

    2. Configure Environment Variables

    Copy the example file:

    cp example.env .env

    docker compose up -d

    You can now open http://localhost:8000/openapi/ to explore your API.
